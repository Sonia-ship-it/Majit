# Gilroy Font Family

Download from: https://www.fontshmonts.com/text-fonts/gilroy/

## Files Needed

Place the following files in this directory:

### Required weights:
- `Gilroy-Regular.woff2` / `.woff` (400)
- `Gilroy-Medium.woff2` / `.woff` (500)
- `Gilroy-SemiBold.woff2` / `.woff` (600)
- `Gilroy-Bold.woff2` / `.woff` (700)
- `Gilroy-ExtraBold.woff2` / `.woff` (800)

## How to Get Fonts

1. Visit the link above
2. Click "Download Gilroy Font" button
3. Extract ZIP file
4. Find the weight variants you need
5. Convert to `.woff2` if needed (use https://transfonter.org/)
6. Rename to match the names above
7. Copy them to this folder

See `FONTS_SETUP.md` in the project root for detailed instructions.

# Super Vanilla Font

Download from: https://www.fontspace.com/super-vanilla-font-f152000

## Files Needed

Place the following files in this directory:
- `SuperVanilla-Regular.woff2`
- `SuperVanilla-Regular.woff`

## How to Get Fonts

1. Visit the link above
2. Click "Download" button
3. Extract ZIP file
4. Find `.woff2` and `.woff` files (or convert from `.ttf`)
5. Rename to match the names above
6. Copy them to this folder

See `FONTS_SETUP.md` in the project root for detailed instructions.
